1- You need to create .env file and put the URI of your database and save it at "MONGO_URI" variable and put your ports 
at variables "PORT1","PORT2","PORT3".
2- The validation done by express-validator and by mongoose validation.
3- Logging done using morgan.
4- Chaching done using Redis.
5- Load balance done using NGINX.
6- Please don't forget to run redis server and nginx server first.
